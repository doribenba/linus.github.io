Linus Torvalds

A legendary programmer, inventor of Git, creator of the Linux kernel, more about him here.

My name is Linus, and I am your god .- Linus Benedict Torvalds

The well-known developer originally came from Helsinki, where he was born on December 28, 1969.

He began programming at an early age, creating example programs from computer magazines.

However, it wasn't until he was 21 that he started his first well-known project: Linux.

Linux is an open-source operating system kernel based on Unix (just like macOS, only open-source). Linux formed the foundation for many distributions (also called distros) such as Ubuntu and Arch (I use Arch btw), while systems like Android are also based on Linux.
But Linux is not his only famous project. In 2005, he created Git, a version control system to support the development of the Linux kernel. It enabled developers to collaborate efficiently and track changes. Later, GitHub emerged from it (I even published my first program there), a platform that uses Git to host code and facilitate collaboration in software development.

In the first commit through Git, he described Git as:

  • the information manager from hell
  • random three letter combination that is pronounceable and not used by Unix
  • mispronunciation of "get"
  • "global information manager" when you are in a good mood
  • "goddam idiotic truckload of sh*t" when it breaks

Linus Torvalds revolutionized software development with Linux and Git, making open-source software accessible and enabling developers worldwide to collaborate efficiently and create their own customized systems.

I thank him.
